This guide helps you set up a development environment for contributing to Memos.

## Prerequisites

- **Go 1.21+** - [golang.org/dl](https://golang.org/dl/)
- **Node.js 18+** - [nodejs.org](https://nodejs.org/)
- **pnpm** - Package manager
- **Git**

## Clone Repository

```bash
git clone https://github.com/usememos/memos.git
cd memos
```

## Backend

```bash
# Build and run
go build -o ./build/memos ./cmd/memos
./build/memos --mode dev --port 8080

# Or with hot reload (install Air first: go install github.com/cosmtrek/air@latest)
air
```

## Frontend

```bash
cd web
pnpm install
pnpm dev
```

The frontend runs at `http://localhost:3000` and proxies API requests to the backend.

## Full Stack

Run both concurrently:

```bash
# Terminal 1 - Backend
air

# Terminal 2 - Frontend
cd web && pnpm dev
```

## Testing

```bash
# Backend tests
go test ./...

# Frontend tests
cd web && pnpm test
```

## Contributing

```bash
# Create feature branch
git checkout -b feature/my-feature

# Commit with conventional commits
git commit -m "feat: add my feature"

# Push and create PR
git push origin feature/my-feature
```
