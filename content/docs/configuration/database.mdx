---
title: Database
description: SQLite, MySQL, and PostgreSQL configuration.
---

Memos supports three database backends:

- `sqlite` (default)
- `mysql`
- `postgres`

## SQLite (default)

If `--driver=sqlite` and no DSN is provided, Memos creates a database file in the data directory:

- `memos_prod.db` (normal mode)
- `memos_demo.db` (demo mode)

You can control the data directory with `--data` or `MEMOS_DATA`.

## MySQL / PostgreSQL

For MySQL or PostgreSQL, set `--driver` and provide a DSN with `--dsn` (or `MEMOS_DSN`). The DSN format follows the standard driver conventions for each database.

```bash
MEMOS_DRIVER=postgres MEMOS_DSN="postgres://user:pass@host:5432/memos" memos
```
