Memos is configured using command-line flags or `MEMOS_*` environment variables.

## Common options

| Flag | Environment | Description | Default |
| --- | --- | --- | --- |
| `--demo` | `MEMOS_DEMO` | Enable demo mode | `false` |
| `--addr` | `MEMOS_ADDR` | Bind address | empty |
| `--port` | `MEMOS_PORT` | Bind port | `8081` |
| `--unix-sock` | `MEMOS_UNIX_SOCK` | Unix socket path | empty |
| `--data` | `MEMOS_DATA` | Data directory | auto |
| `--driver` | `MEMOS_DRIVER` | Database driver | `sqlite` |
| `--dsn` | `MEMOS_DSN` | Database DSN | empty |
| `--instance-url` | `MEMOS_INSTANCE_URL` | Public instance URL | empty |

## Example

```bash
MEMOS_DRIVER=sqlite MEMOS_DATA=/var/opt/memos memos --port 8081
```

See the sections below for database, storage, authentication, and security settings.
