Memos stores memo attachments using instance storage settings. Supported storage types include:

- Database storage
- Local filesystem storage
- S3-compatible object storage

## Configure storage

Storage settings are managed by administrators in the app settings or via the Instance Settings API.

Key options include:

- Storage type
- File path template for local storage
- Upload size limit
- S3 configuration (endpoint, region, bucket, access keys)

See the [API reference](/docs/api) for the exact storage setting fields.
