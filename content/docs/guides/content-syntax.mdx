Memos supports Markdown formatting following the CommonMark and GitHub Flavored Markdown (GFM) specifications. This guide covers the most commonly used syntax.

## Text Formatting

```markdown
**Bold text**
*Italic text*
~~Strikethrough~~
`Inline code`
```

**Result:** **Bold text**, *Italic text*, ~~Strikethrough~~, `Inline code`

## Headers

```markdown
# Heading 1
## Heading 2
### Heading 3
```

## Lists

**Bullet list:**
```markdown
- First item
- Second item
  - Nested item
```

**Numbered list:**
```markdown
1. First item
2. Second item
3. Third item
```

**Task list:**
```markdown
- [x] Completed task
- [ ] Pending task
```

## Links and Images

**Links:**
```markdown
[Link text](https://example.com)
https://usememos.com
```

**Images:**
```markdown
![Alt text](https://example.com/image.png)
```

## Code

**Inline:** Use backticks for inline `code`.

**Code blocks:**
````markdown
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```
````

**Supported languages:** `javascript`, `python`, `go`, `bash`, `sql`, `json`, `yaml`, `typescript`, `html`, `css`, and many more.

## Tables

```markdown
| Name  | Role      | Status |
|-------|-----------|--------|
| Alice | Developer | Active |
| Bob   | Designer  | Active |
```

**Alignment:**
```markdown
| Left  | Center | Right |
|:------|:------:|------:|
| Text  | Text   | Text  |
```

## Quotes

```markdown
> This is a quoted text.
>
> It can span multiple lines.
```

## Horizontal Line

```markdown
---
```

## Tags

Organize your memos with tags:

```markdown
#work #project #urgent
#personal #ideas
```

**Tag tips:**
- Use lowercase for consistency
- Use hyphens for multi-word tags: `#project-alpha`
- Keep it simple: 3-5 tags per memo

## Quick Reference

| Syntax | Result |
|--------|--------|
| `**bold**` | **bold** |
| `*italic*` | *italic* |
| `~~strike~~` | ~~strike~~ |
| `` `code` `` | `code` |
| `[text](url)` | [text](url) |
| `![alt](url)` | image |
| `# heading` | heading |
| `- item` | bullet |
| `1. item` | numbered |
| `- [ ] task` | checkbox |
| `> quote` | quote |
| `---` | horizontal line |
| `#tag` | tag |

## Additional Features

**Collapsible sections:**
```html
<details>
<summary>Click to expand</summary>
Content here.
</details>
```

**Keyboard shortcuts:**
```html
<kbd>Ctrl</kbd> + <kbd>S</kbd>
```
