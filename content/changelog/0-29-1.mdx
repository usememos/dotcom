> This patch release tightens up memo rendering and media previews. It improves task-list layout in markdown, reads standard meta description tags for link previews, and restores video attachment posters on mobile devices.

## Bug Fixes

- **Markdown task lists** - Task item content now stays in a single grid column, keeping checkbox rows aligned with the rest of the memo body.
- **Link preview descriptions** - Link metadata parsing now supports `<meta name="description">`, improving preview summaries for sites that use standard description metadata.
- **Mobile video posters** - Video attachment posters now render correctly on mobile, making media attachments easier to identify before playback.
