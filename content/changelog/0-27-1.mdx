> This patch release fixes user resource naming edge cases introduced around identity provider handling and ensures release artifacts carry the correct build version.

## Bug Fixes

- **Mixed-case user names** - User resource names now support mixed-case values.
- **Identity provider subjects** - User resource names can now be UUIDv4 values from identity provider `sub` claims.
- **Release artifacts** - Build versions are now injected into release artifacts.
